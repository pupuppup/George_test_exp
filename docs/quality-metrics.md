# Quality Metrics

| Metric                        | Target      |
|-------------------------------|-------------|
| Crash-free call action        | >99.9%      |
| Debounce success              | (#tap events) == (#dialer launches) |
| Localisation coverage         | EN / UA / SK |

- Debounce window: 1500ms
- ACTION_DIAL (no CALL_PHONE permission)
- Offline disables call button, shows banner
